# 決算書取込ガイド（毎年の更新手順）

東京ブラインド工業株式会社の「決算報告書」PDFを毎年取り込み、
「📋 決算書分析」タブの比較分析に反映するための手順とルールをまとめたもの。

## 対象PDF

決算報告書（表紙・貸借対照表・損益計算書・販売費及び一般管理費内訳書・
製造原価報告書・株主資本等変動計算書・注記表で構成される一式）。

## 取込手順

1. アプリの「📋 決算書分析」タブ → 左側「📄 PDFを取り込む」から該当年度の決算報告書PDFをアップロード
2. 「🤖 Geminiで解析・取り込み」を実行
   - `gemini_processor.py` の `EXTRACTION_PROMPT` が以下を自動抽出する
3. 取り込み後、左側の「取り込み済み決算書」一覧に当該決算期が追加されていることを確認
4. 比較したい決算期を選択すると、右側に貸借対照表・主要指標の比較表とグラフが表示される

## 抽出・登録される項目

| 区分 | 項目 | PDF上の場所 |
|---|---|---|
| 決算期 | `fiscal_year`（例: 2025年7月期） | 表紙の「第◯期」「自◯至◯」から決算期末の年月で算出 |
| 貸借対照表 | `total_assets` / `total_liabilities` / `net_assets` | 資産の部合計 / 負債の部合計 / 純資産の部合計 |
| 損益計算書 | `sales` / `cost_of_goods` / `gross_profit` | 売上高合計 / 売上原価 / 売上総利益金額 |
| 損益計算書 | `operating_profit` / `ordinary_profit` / `net_profit` | 営業利益金額 / 経常利益金額 / 当期純利益金額 |
| 損益計算書 | `non_operating_income` / `non_operating_expenses` | 営業外収益合計 / 営業外費用合計 |
| 損益計算書 | `extraordinary_income` / `extraordinary_loss` | 特別利益合計 / 特別損失合計 |
| （表示時に算出） | 税引前当期純利益 | 経常利益 + 特別利益 − 特別損失 |

## 固定費／変動費の区分ルール

「販売費及び一般管理費内訳書」の各項目を以下の基準で分類する
（`gemini_processor.py` の `FIXED_COST_ITEM_KEYS` で定義）。

- **固定費** とする項目:
  - 荷造発送費
  - 会議費
  - 交際費
  - 支払手数料
- **変動費**: 上記以外の販管費項目すべて（販管費合計 − 固定費）

```
固定費   = 荷造発送費 + 会議費 + 交際費 + 支払手数料
変動費   = 販売費及び一般管理費合計 − 固定費
```

この区分を変更したい場合は `gemini_processor.py` の `FIXED_COST_ITEM_KEYS` を編集する。
